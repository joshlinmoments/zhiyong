# 智勇網站 — Present（講道簡報）模式設計規格

日期：2026-06-25
範圍：把現有「展示」模式改造成可左右切換的簡報；編輯模式可定義換頁、編排註解頁與自訂頁。

## 目標（使用者已確認）

1. Present 模式像簡報：一次一頁、左右切換。
2. 編輯模式可手動插入「換頁」決定每頁邊界。
3. 編輯模式有「簡報編排」清單，可挑選並排序：內文頁＋註解頁＋新增頁；新增頁支援 PPT 式自由拖曳排版。

## 已確認的決策

- **換頁**：手動換頁點（在區塊編輯器插入）。
- **註解進簡報**：在「簡報編排」清單手動挑選並排序。
- **新增頁排版**：真正拖曳定位（像 PPT，元素可自由移動、縮放）。
- **簡報範圍**：以「目前章節」為一份（與編輯模式一致）。
- **放映操作**：鍵盤 ←→、點畫面左右、全螢幕、右下角頁碼、ESC 離開。
- **內容過長**：該頁可上下捲動。

## 資料模型（Firebase RTDB）

### 換頁
- 新增區塊 level：`換頁`。以一般 block 儲存（`level:'換頁'`, `text:''`）。
- 閱讀/`deriveData()` 忽略換頁 block；編輯器中渲染成虛線分隔「✂ 換頁」。
- 一個章節依換頁切成數個「內文頁（content segment）」，segment 以其第一個 block key 識別。

### Deck（每章一份）
`content/decks/{chapterKey}/{slideId}` = `{ order:number, kind, ... }`
- `kind:'content'` → `{ ref:<segmentStartBlockKey> }`（自動同步，內文有增減時自動補/移除）
- `kind:'anno'` → `{ annKey }`
- `kind:'free'` → `{ els:{ elId:{ type:'text'|'image'|'video', x,y,w,h, ...payload } } }`（座標用畫布百分比，畫布固定 16:9）

### 自動對齊（reconcile）
載入 deck 時，依目前換頁切出 segments：
- 缺少 content slide 的 segment → 自動補一張（插在前一段 content slide 之後）。
- segment 已不存在 → 移除其 content slide。
- anno / free slide 保持使用者擺放的位置與順序（order 欄位）。

## UI

### 區塊編輯器
- level 下拉新增「換頁」；換頁 block 顯示為虛線分隔列，不顯示文字框。
- （或提供獨立「插入換頁」按鈕，視實作便利性。）

### 簡報編排 Organizer（編輯模式）
- 入口：編輯模式底部工具列新增「🎞 簡報編排」。
- 左側：本章投影片清單（縮圖＋類型標籤），可拖曳排序；content 頁顯示首行、可隱藏；anno/free 可刪除。
- 「＋ 新增空白頁」→ 建立 free slide 並開啟畫布編輯器。
- 「本章註解」面板：列出本章註解，點「加入簡報」插入一張 anno slide。

### PPT 式自由排版（free slide 畫布編輯器）
- 固定 16:9 畫布；元素：文字框 / 圖片 / 影片。
- 拖曳移動、控點縮放、點選編輯、工具列新增元素 / 刪除 / 上下層。
- 座標以百分比存，確保放映縮放一致。

### 放映 Playback（展示）
- 全螢幕單頁檢視；←/→、點畫面左右切頁、全螢幕鈕、右下角「3 / 12」、ESC 離開。
- content 頁顯示該 segment 內文（過長可捲動）；anno 頁套用乾淨置中版型；free 頁等比縮放畫布。

## 分階段交付

- **Phase 1**：換頁機制 ＋ 內文分頁放映（←→ / 全螢幕 / 頁碼 / ESC）。先讓基本簡報可用。
- **Phase 2**：簡報編排清單 ＋ 註解頁（挑選、排序、隱藏 content 頁）。
- **Phase 3**：PPT 式自由排版新頁（拖曳定位、縮放）。

每階段獨立 push、可單獨驗收。

## 不在範圍（暫不做）
- 跨章節的整本大簡報。
- 動畫轉場、簡報主題切換。
- 多人即時協作的鎖定（沿用現有 last-write-wins）。
